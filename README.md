# weather-app
Aplicativo que tem o objetivo de mostrar a previsão do tempo para os próximos dias

Instruções:

Instanlando o ionic e o cordova
npm install ionic cordova -g

<h1>Instalando as dependencias</h1>
<p>npm install</p>
<p>bower install</p>

<h1>Testando no browser</h1>
<p>ionic serve</p>

<h1>Testando no android</h1>
ionic build android
ionic run android
